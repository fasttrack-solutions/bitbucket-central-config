# bitbucket-central-config

Grab all members:
https://api.bitbucket.org/2.0/workspaces/{workspace}/members

Grab all repos:
https://api.bitbucket.org/2.0/repositories/{workspace}

Get a single repo:
https://api.bitbucket.org/2.0/repositories/{workspace}/{repo}


Get Default reviews
https://api.bitbucket.org/2.0/repositories/{workspace}/{repo}/default-reviewers

Get Branch restrictions
https://api.bitbucket.org/2.0/repositories/{workspace}/{repo}/branch-restrictions


Get Branch Model
https://api.bitbucket.org/2.0/repositories/{workspace}/{repo}/branching-model
